# Real-time Chat Application

A modern real-time chat application built with Next.js, Socket.IO, and Express.js.

## Features

- Real-time messaging
- File sharing (images)
- Typing indicators
- User join notifications
- Modern UI with Tailwind CSS
- Responsive design
- Real-time message broadcasts
- User avatars with initials

## Tech Stack

### Frontend
- Next.js 14.2
- React 18
- Socket.IO Client
- Tailwind CSS
- PostCSS

### Backend
- Express.js
- Socket.IO
- Node.js

## Project Structure

```
├── client/               # Frontend application
│   ├── app/             # Next.js app directory
│   ├── components/      # React components
│   ├── assets/         # Static assets
│   └── public/         # Public files
└── server/             # Backend application
```
